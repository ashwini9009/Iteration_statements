 Programs Included

- Prints all numbers from 1 up to a given number N
- Uses for loop
---
 2️⃣ Print Sum of N Numbers
- Calculates the sum of numbers from 1 to N
- Uses loop 

---

 3️⃣ Reverse a Number
- Reverses the digits of a given number
- Uses while loop, % and / operators

**Example:**  
Input: 123  
Output: 321

---

 4️⃣ Factorial of a Number
- Calculates factorial using iteration
- Formula: n! = n × (n-1) × ... × 1

**Example:**  
Input: 5  
Output: 120

---

 5️⃣ Count Digits in a Number
- Counts total digits in a given number
- Uses repeated division by 10

**Example:**  
Input: 12345  
Output: 5

---

6️⃣ Fibonacci Series
- Prints Fibonacci sequence up to N terms
- Each number is the sum of previous two numbers

**Series Example:**  
0 1 1 2 3 5 8 ...
